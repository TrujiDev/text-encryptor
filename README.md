# Encriptador y Desencriptador

Este es un pequeño proyecto de encriptación y desencriptación de texto desarrollado con HTML, CSS y JavaScript.

### Descripción

El código consta de tres archivos principales:

- `index.html`: Contiene la estructura HTML para la interfaz de usuario.
- `style.css`: Define los estilos CSS para dar formato y diseño a la interfaz.
- `main.js`: Contiene el código JavaScript que realiza las funciones de encriptación y desencriptación.

### Funcionalidades

El proyecto ofrece las siguientes funcionalidades:

1. **Encriptación**: Convierte un texto ingresado en la interfaz a una forma encriptada según un conjunto de reglas definidas.
2. **Desencriptación**: Permite desencriptar un texto encriptado de acuerdo con las reglas inversas de encriptación.
3. **Copia al Portapapeles**: Facilita la copia del texto encriptado o desencriptado al portapapeles del dispositivo.

### Reglas de Encriptación

El código utiliza un conjunto de reglas de encriptación predefinidas, donde cada letra especificada es reemplazada por otra combinación de letras. Las reglas de encriptación actualmente definidas son:

- `e` por `enter`
- `i` por `imes`
- `a` por `ai`
- `o` por `ober`
- `u` por `ufat`

### Uso

1. Clona o descarga este repositorio en tu máquina local.
2. Abre el archivo `index.html` en un navegador web.
3. Ingresa un texto en el campo proporcionado.
4. Haz clic en el botón "Encriptar" para obtener el texto encriptado o en "Desencriptar" para revertir la encriptación.
5. Puedes hacer clic en el botón "Copiar" para copiar el texto resultante al portapapeles.

### Notas

- Este proyecto está diseñado para encriptar y desencriptar textos que consisten solo en letras minúsculas sin acentos.
- Se proporciona una advertencia en la interfaz para recordar al usuario sobre las restricciones de entrada.

¡Disfruta encriptando y desencriptando tus mensajes de texto con este sencillo proyecto!
